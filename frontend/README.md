# FocusFrame Landing

Next.js landing page for FocusFrame, the free macOS screen recorder and guided editor.

## Local Development

```bash
bun install
bun run dev
```

Open `http://localhost:3000`.

## Checks

```bash
bun run lint
bun run build
```

## Assets

Optimized website assets live in `public/`. Raw local recordings stay outside this app and are ignored by the root `.gitignore`.
